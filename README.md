# Manuais

Catálogo estático dos manuais do e-Cidade e materiais relacionados.

- Os arquivos originais ficam em `2014/`, `2018/` e `arquivos/`.
- Rode [`scripts/build_site.py`](scripts/build_site.py) para gerar o site em `site/`.
- O workflow do GitHub Pages publica a pasta `site/`.
- Para incluir novos manuais, copie os arquivos para uma dessas pastas e rode o build de novo.
